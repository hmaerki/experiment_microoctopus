## Protype Baord

### Precondition

On schematics:
 * BP101: Gnd

On pcb:
 * Set GND point: Array 26x20
 * `Selection Filter`: Only check `Pads`
 * Select all pads which are NOT GND
 * `Properties -> Basic Properties -> Net: <empty>`
